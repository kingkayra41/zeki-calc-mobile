# 📱 Zeki Calculator Pro (Mobile)

[TR] 9. Sınıf bir geliştirici tarafından Python & Flet kullanılarak yapılmış, akıllı matematik motoruna sahip mobil hesap makinesi.
[EN] A mobile calculator with a smart math engine, built by a 9th-grade developer using Python & Flet.

---

## ✨ Özellikler / Features

### 🇹🇷 Türkçe
* **Akıllı Çarpma:** `3π` veya `(2+2)5` gibi ifadeleri otomatik anlar ve çarpar.
* **Çift Dil Desteği:** Tek tuşla Türkçe ve İngilizce arasında geçiş yapın.
* **Modern Tasarım:** Karanlık mod (Dark Mode) ve şık butonlarla göz yormaz.
* **Matematiksel Güç:** `e`, `π` ve üslü sayılar (`^`) desteği.

### 🇺🇸 English
* **Smart Multiplication:** Automatically recognizes and multiplies expressions like `3π` or `(2+2)5`.
* **Dual Language Support:** Switch between Turkish and English with a single tap.
* **Modern Design:** User-friendly Dark Mode interface with elegant buttons.
* **Mathematical Power:** Supports `e`, `π`, and exponents (`^`).
